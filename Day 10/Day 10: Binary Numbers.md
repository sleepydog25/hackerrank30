# Day 10: Binary Numbers

## Input Format
A single integer n.

## Output Format
Print a single base-10 integer that denotes the maximum number of consecutive 1's in the binary representation of n.

### Sample Input 1
5
### Sample Output 1
1
### Sample Input 2
13
### Sample Output 2
2
